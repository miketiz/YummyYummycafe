# Project Pivot: YummyYummy Cafe

## Overview

YummyYummy Cafe เริ่มต้นจากแนวคิดการทำเว็บไซต์ร้านเบเกอรี่/คาเฟ่สำหรับแสดงเมนูและรับออเดอร์ออนไลน์ แต่ระหว่างการพัฒนา พบว่าความต้องการจริงของร้านไม่ได้จบแค่การให้ลูกค้ากดสั่งซื้อได้เท่านั้น เจ้าของร้านยังต้องการระบบที่ช่วยจัดการงานหลังบ้าน เช่น ตรวจสอบออเดอร์ แจ้งเตือนคำสั่งซื้อ สรุปยอดขาย ติดตามสถานะการชำระเงิน และเตรียมออเดอร์สำหรับจัดส่ง

โปรเจกต์จึง pivot จากเว็บไซต์สั่งอาหารธรรมดา ไปเป็นระบบจัดการออเดอร์อัจฉริยะสำหรับร้านขนาดเล็ก โดยเพิ่ม AI Chat, Admin Dashboard, Telegram Back-office Bot, Google Sheet Order Export และระบบพิมพ์ใบออเดอร์สำหรับแปะถุงหรือกล่องสินค้า

## Before Pivot

- เว็บไซต์แสดงเมนูและรับออเดอร์ออนไลน์
- ลูกค้าต้องเลือกสินค้าและกรอกข้อมูลเองเป็นหลัก
- ข้อมูลออเดอร์ถูกบันทึกในระบบ แต่ยังช่วยงานหลังบ้านได้จำกัด
- เจ้าของร้านยังต้องตรวจสอบออเดอร์และสรุปยอดด้วยตัวเองหลายขั้นตอน

## Problem Found

หลังจากมองในมุมการใช้งานจริงของร้าน พบปัญหาหลักคือระบบสั่งซื้ออย่างเดียวไม่เพียงพอสำหรับการทำงานประจำวันของเจ้าของร้าน เพราะร้านต้องการ:

- รับออเดอร์ใหม่ให้รู้ทันที
- เห็นออเดอร์ที่ยังไม่จัดส่ง
- ตรวจสอบออเดอร์ที่ยังไม่ชำระเงิน
- เปลี่ยนสถานะออเดอร์ได้ง่าย
- มีข้อมูลยอดขายสำหรับสรุปรายวัน
- มีใบออเดอร์สำหรับแปะถุงหรือกล่องสินค้า
- มีช่องทางหลังบ้านที่ถามข้อมูลได้เร็วโดยไม่ต้องเปิดหน้าเว็บตลอดเวลา

## Pivot Decision

โปรเจกต์จึงเปลี่ยนทิศทางจาก:

> เว็บไซต์ขายเบเกอรี่

เป็น:

> AI-powered order management system สำหรับร้านคาเฟ่ขนาดเล็ก

แนวทางใหม่นี้เน้นให้ระบบเป็นผู้ช่วยเจ้าของร้าน ไม่ใช่แค่หน้าร้านออนไลน์ โดยครอบคลุมทั้งฝั่งลูกค้าและฝั่งหลังบ้าน

## After Pivot

- ลูกค้าสามารถถามเมนูและสั่งซื้อผ่าน AI Chat หน้าเว็บได้
- ระบบเพิ่มสินค้าเข้าตะกร้าและสร้างออเดอร์ได้จากแชท
- ออเดอร์ถูกบันทึกลง Supabase Database
- ออเดอร์ถูกส่งเข้า Google Sheet เพื่อใช้ดูรายงานและสำรองข้อมูล
- Telegram แจ้งเตือนทันทีเมื่อมีออเดอร์ใหม่
- Telegram Back-office Bot ใช้ถามข้อมูลหลังบ้านได้ เช่น ออเดอร์ค้างส่ง ยอดขายวันนี้ และออเดอร์ที่ยังไม่ชำระ
- Admin Dashboard ใช้จัดการเมนู ออเดอร์ สถานะการจัดส่ง และสถานะการชำระเงิน
- ระบบสามารถพิมพ์ใบออเดอร์สำหรับแปะสินค้าได้

## Key Improvements

### Customer Side

- มี AI Chat ช่วยแนะนำเมนู
- ลูกค้าสั่งซื้อผ่านแชทได้
- ตะกร้าสินค้าและฟอร์มสั่งซื้อใช้งานง่ายขึ้น
- ลูกค้าสามารถตรวจสอบสถานะออเดอร์ได้

### Back-office Side

- Admin เห็นออเดอร์ทั้งหมดในที่เดียว
- มีตัวกรองสถานะออเดอร์และสถานะชำระเงิน
- เปลี่ยนสถานะออเดอร์และสถานะชำระเงินได้
- พิมพ์ใบออเดอร์สำหรับจัดเตรียมสินค้าได้
- Telegram ใช้เป็นผู้ช่วยหลังบ้านได้

### Data and Automation

- บันทึกข้อมูลลง Database เป็นแหล่งข้อมูลหลัก
- Sync ออเดอร์ไป Google Sheet
- Sync การเปลี่ยนสถานะจาก Admin/Telegram ไป Google Sheet
- ส่งรายงานยอดขายรายวันผ่าน Telegram
- แจ้งเตือน Telegram เมื่อมีออเดอร์ใหม่

## Value After Pivot

หลัง pivot แล้ว ระบบมีคุณค่ามากขึ้น เพราะไม่ได้เป็นเพียงเว็บไซต์หน้าร้าน แต่กลายเป็นเครื่องมือช่วยเจ้าของร้านทำงานจริง ตั้งแต่รับออเดอร์ แจ้งเตือนหลังบ้าน จัดการสถานะ ดูรายงานยอดขาย ไปจนถึงเตรียมสินค้าเพื่อจัดส่ง

ระบบนี้เหมาะกับร้านเล็กที่เจ้าของร้านต้องทำหลายหน้าที่พร้อมกัน และต้องการใช้ AI/automation มาช่วยลดงานซ้ำในแต่ละวัน

## Pivot Summary

YummyYummy Cafe pivot จากเว็บไซต์ขายเบเกอรี่ธรรมดา ไปเป็นระบบจัดการออเดอร์อัจฉริยะสำหรับร้านคาเฟ่ขนาดเล็ก โดยใช้ AI Chat, Telegram Bot, Google Sheet และ Admin Dashboard เพื่อช่วยทั้งลูกค้าและเจ้าของร้านในกระบวนการสั่งซื้อและจัดการหลังบ้าน

สรุปสั้น ๆ:

> From online bakery storefront to AI-powered cafe order operations system.
